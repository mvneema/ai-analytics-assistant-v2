# AI Analytics Assistant v2.0

AI-powered analytics assistant that converts natural language questions into SQL queries with interactive visualizations and exportable reports.

## 🚀 Features

- **Natural Language to SQL** - Ask questions in plain English, get SQL queries
- **Interactive Charts** - Bar and line charts with Recharts
- **AI Insights** - Claude AI-powered analysis and recommendations
- **Query History** - Persistent history of last 10 queries (localStorage)
- **Export to HTML** - Download complete reports with embedded charts
- **Highlight Boxes** - Color-coded key takeaways
- **Demo Mode** - Works without API key using pre-built responses
